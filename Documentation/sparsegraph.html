Sparse Graph Methods

__binarySearch__(row, col)¶

Binary search for inserting and removing elements in a list.

This function is used to search for the position for orderly insertion or removal of an element in a list. The function return is of the type 2-tuple, where the first position is a value of 1 or 0 that indicates whether or not the searched element is present in the list, respectively. The second position of the 2-tuple indicates the position of / for the element.

Parameters

  • row – List to fetch it.

  • col – Element to be searched.

Returns

A 2-tuple that contains if the element existis or not and their position.

__findDiagonal__()¶

Finds the position of each element within its own adjacency list.

This function is used to find the position of the adjacencies of each element for itself within its own adjacency lists. The position found is added in a vector called n positions diagonal, in the index corresponding to the position of each element in the input set. It is essential for quick access to the arc weight of each element.

__indexCorrector__(index)¶

Correction of the indexes of the adjacencies after a removal.

This function is used to correct the values ​​of the first position of the 2-tuples that represent the edges within the adjacency lists of all vertices. Values ​​greater than the index of the vertex that was removed are decremented by 1, in order to correct the resulting inconsistency after removal.

Parameters

index – Reference index to the correction.

__insertEdge__(row, col, weight)¶

Insertion of an edge between a pair of vertices.

This function is used for inserting an edge between a pair of graph vertices. Uses the __binarySearch__ function to find the correct position for an ordered insertion. The insertion is undirected. Therefore, two edges are created at each insertion. The first is from row to col and the second is your mirror.

Parameters

  • row – First vertex of the pair.

  • col – Second vertex of the pair.

  • weight – Weight of the edge between the pair of vertices.

__removeEdge__(row, col)¶

Removing an edge between a pair of vertices.

This function is used to remove the edge between a pair of vertices of the graph. It uses the function __binarySearch__ to check if there is an edge or not and find the position of the edge to be removed within the list of adjacencies of both vertices, considering that the removal is also non-directed.

Parameters

  • row – Vertex of origin of the edge.

  • col – Target apex of the edge.

Returns

True or false if the edges was or wasn’t removed.

__removeVertex__(index)¶

Removing a vertex from the input set.

This function is used to remove a vertex from the input set. Vertices are represented only by their adjacency lists. The first step of vertex removal is the removal of all edges that connect with it and also all mirror edges. After that, the list of element adjacencies is removed. After the removal, it is necessary to make a correction of the indices higher than the vertex removed within the lists of adjacencies of all elements. This task is done with the function __indexCorrector__.

Parameters

index – Index of the vertex to be removed.
