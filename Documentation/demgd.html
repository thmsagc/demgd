__applyKernelAndNormalize__()¶

Application of core function and normalization on edge weights.

This function is responsible for geometrizing the graph and transforming the set of edges into a Markov chain. First, the core function is applied to each edge weight, represented by the second position of the 2-tuples in the adjacency lists. The new values ​​are added to the divisor variable, which will be the normalization factor. Then, all values ​​are scanned again to divide each one by the normalization factor.

__bucketsVerify__(maxPerBucket)¶

Check if any bucket has exceeded its maximum capacity.

This function is responsible for checking if any bucket has exceeded the maximum capacity of elements. If any bucket has exceeded its maximum capacity, this function returns the Boolean value False. If all buckets are at maximum capacity, this function returns the Boolean value True.

Parameters

numBuckets – Number of buckets for allocating elements.

Returns

True or False if the verification was successful or not.

__buildBuckets__()¶

Construction of buckets.

This function is responsible for allocating the elements in their respective buckets. Use the __pertinenceDegree__ function to calculate the bucket index that each element should be allocated to.

__computeAverage__()¶

Computation of the average of the importance values ​​of the elements.

This function is responsible for computing the average of the elements’ importance. First, the main diagonal of the adjacency matrix is ​​added by multiplying each value of the diagonal by the number of vertices in the star of each element. Then, the result of this sum is divided by the number of vertices.

Returns

The value of the average of the importances.

__computeMinMax__()¶

Computation to find the minimum and maximum points of the set and entry.

This function is responsible for finding the minimum and maximum points of the input set. These values ​​make up the square convex closure of the input set.

__computeStandardDeviation__()¶

Computation of the standard deviation of the importance values ​​of the elements.

This function is responsible for computing the standard deviation of the importance values ​​of the elements. This value is the result of the square root of the variance of the importance values ​​of the elements.

Returns

The value of Standard Deviation of the importances.

__computeStatistics__()¶

Computation to calculate the mean, variance and standard deviation of the values ​​of the importance of the elements.

This function is an auxiliary function, which calls the functions responsible for calculating the mean, variance and standard deviation of the importance values ​​of the elements in this exact order.

__computeVariancy__()¶

Computation of the standard deviation of the importance values ​​of the elements.

This function is responsible for computing the variance of the importance values of the elements. First it is subtracted from the average of the importance values of the importance of each element. The result of this subtraction is added to a variable called total. Finally, the total value of the sum of the subtractions is divided by the number of vertices.

Returns

The value of the Variancy of the importances.

__createGraphKNN__(k, propagation=0)¶

Creation of the graph for geometric representation of the input set.

This function is responsible for creating the graph that represents the geometric structure built from the input set. First, the k-neighbors closest to each element are found in their own bucket and a number represented by the neighbor propagation parameter. Then, edges are created between each element and its closest k-neighbors. Finally, the __applyKernelAndNormalize__ and __computeStatistics__ functions are called.

Parameters

  • k – Number of k-closest neighbors that each vertex must connect to.

  • propagation – Number of neighboring buckets that must be traversed to compute the nearest k-neighbors.

__mergeBuckets__(maxPerBucket)¶

Union of buckets that contain few or no elements.

This function is responsible for joining buckets with a low number of elements. For each bucket, it is checked that the connection with the following buckets does not exceed the maximum capacity of elements per bucket and joins the buckets until the connection with the next bucket does not exceed the maximum capacity of elements per bucket.

Parameters

maxPerBucket – Maximum number of elements per bucket.

The pertincence degree of a xk inputed.
