Execution.DeterministicEMGD(inputSet, percentage=0.75, maxPerBucket=25, k=5, propagation=1, multiplier=1, distanceMetric='euclidean')¶

Encapsulation of the Deterministic Extended Markovian Geometric Diffusion method.

This function is responsible for encapsulating the entire DEMGD method, so that the user needs to make only one function call to execute the entire method.

Parameters

  • inputSet – Set of input to be reduced.

  • percentage – Percentage of the number of elements to be maintained from the input set.

  • maxPerBucket – Maximum number of elements per bucket.

  • k – Number of edges that each element must have.

  • propagation – Number of neighboring back and forth buckets that must be traversed for the computation of the nearest k-neighbors.

  • multiplier – Multiplier for standard deviation.

  • distanceMetric – Metric of distance to be used.

Execution.buildSetFromTxt(filename)¶

Reading a TXT file as a List.

This function is responsible for reading a set from a user-specified txt file. Each element of the set must be represented by a line from the file. This function returns a list of elements.

Parameters

filename – Name of the file to be read.

Returns

A list that represents a set.
